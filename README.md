# 우리반 이음

학생의 월별 자기평가와 교우관계 응답을 바탕으로 담임교사가 지원 신호, 관계 변화, 관찰 포인트를 근거와 함께 확인하는 반응형 웹 MVP입니다.

## 실행

별도의 설치 과정이 없습니다. `index.html`을 브라우저에서 열거나 로컬 서버로 실행하세요.

```powershell
python -m http.server 4173
```

브라우저에서 `http://127.0.0.1:4173`에 접속합니다.

## 구현 범위

- 교사 홈: 참여율, 즉시 확인, 이번 주 확인, 긍정 변화
- 설문 관리: 참여 링크, 제출 현황, 학생 설문 3단계 미리보기
- 분석 대시보드: 시점별 필터, 근거 원문, 복수 가설, 관찰 제안
- 관계 분석: 추정 관계망, 양극화와 비대칭 지표, 응답률 안전장치
- 관찰 기록: 예정/관찰 중/확인 완료 흐름과 브라우저 로컬 저장
- 모바일·태블릿·PC 반응형 화면
- 홈페이지 내부 학생 설문: 학급 명단에서 이름 선택, 자기평가 5개, 친구 관찰, 교우관계, 도움 요청
- Supabase PostgreSQL DB 응답 누적 저장
- 교사 전용 접근 코드로 제출 현황·원문 확인 및 명단 동기화

## 안전 원칙

학생 화면은 공개 키로 명단 조회와 새 응답 제출만 할 수 있으며, 기존 응답은 조회할 수 없습니다. 교사 화면의 응답 조회와 명단 변경은 교사용 DB 접근 코드가 필요합니다. 응답은 제출 횟수와 기간 제한 없이 매번 별도 기록으로 누적되며 `survey_month`로 설문 기준 월을 구분합니다. 실제 적용 전에는 보존·삭제 정책 및 학교·교육청 개인정보 지침을 검토해야 합니다.

DB 구조와 권한 함수는 `supabase/schema.sql`에 정의되어 있습니다. 이 파일의 `__TEACHER_DB_CODE__` 자리에는 배포 시에만 교사용 코드를 넣으며, 실제 코드는 Git 저장소에 커밋하지 않습니다.
