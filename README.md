# 우리반 이음

학생의 월별 자기평가와 교우관계 응답을 바탕으로 담임교사가 지원 신호, 관계 변화, 관찰 포인트를 근거와 함께 확인하는 반응형 웹 MVP입니다.

## 실행

별도의 설치 과정이 없습니다. `index.html`을 브라우저에서 열거나 로컬 서버로 실행하세요.

```powershell
python -m http.server 4173
```

브라우저에서 `http://127.0.0.1:4173`에 접속합니다.

## 구현 범위

- 교사 홈: 참여율, 즉시 확인, 이번 주 확인, 긍정 변화
- 설문 관리: 참여 링크, 제출 현황, 학생 설문 3단계 미리보기
- 분석 대시보드: 시점별 필터, 근거 원문, 복수 가설, 관찰 제안
- 관계 분석: 추정 관계망, 양극화와 비대칭 지표, 응답률 안전장치
- 관찰 기록: 예정/관찰 중/확인 완료 흐름과 브라우저 로컬 저장
- 모바일·태블릿·PC 반응형 화면
- 홈페이지 내부 학생 설문: 학급 명단에서 이름 선택, 자기평가 5개, 친구 관찰, 교우관계, 도움 요청
- Google Apps Script + 교사 소유 Spreadsheet 응답 저장
- 교사 화면 제출 현황·원문 확인 및 명단 동기화

## 안전 원칙

Apps Script를 설정하지 않으면 응답은 같은 브라우저의 로컬 저장소에만 보관되는 데모 모드로 동작합니다. 실제 학급 운영은 [Google Apps Script 연결 안내](gas/README.md)에 따라 교사 소유 Spreadsheet를 연결해야 합니다. 실제 적용 전에는 교사 인증, 권한 통제, 감사 로그, 보존·삭제 정책 및 학교·교육청 개인정보 지침 검토가 필요합니다.
