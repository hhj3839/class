# 우리반 이음 v1.1.0-rc.2

릴리스 후보 기준일: 2026-07-20  
배포 주소: https://hhj3839.github.io/class/

## 기준

이 버전은 `v1.1.0-rc.1`의 전체 기능을 포함하고, 실제 배포된 학생 참여 링크를 이용한 제출 전 모의시험 결과를 반영한다. 테스트 응답은 운영 데이터베이스에 제출하지 않았다.

## RC1 이후 변경

- 학생 이름의 마지막 글자 받침에 따라 관계 선택 문구의 `과의/와의` 조사를 자동 선택
- `김준후와의 관계`, `마채원과의 관계` 형태를 실제 Pages에서 확인
- `student.js` 캐시 버전을 갱신해 수정 코드가 즉시 반영되도록 조치

## 실제 배포 확인

- 안전한 학생 참여 토큰으로 진입
- 학생 명단 21명 로딩
- 학생 이름 선택과 설문 시작
- 1번부터 9번까지 고정된 문항 순서
- 2번과 5번의 확정 원문
- 8번 본인 스마일 아이콘과 친구별 관계 선택
- 제출 직전 확인 화면과 제출 버튼
- 테스트 응답 미제출

## 자동 검증

- Node 자동 테스트 56개 통과
- 로그인 전 교사용 개인정보 비노출
- 교사용 개인정보의 `localStorage` 저장 금지
- 다른 교사·학급 접근 권한 검사
- 학생 참여 링크의 임의 토큰 사용
- 파일럿 점검 보고서의 개인정보 제외

## 남은 승인 조건

정식 `v1.1.0` 전환 전 다음 현장 시험이 필요하다.

1. 서로 다른 태블릿 3~5대에서 QR 접속
2. 화면 회전·잠금·네트워크 재연결 후 작성 내용 복구
3. 실제 동시 제출과 교사 제출 현황 반영
4. 학교 기기에서 AI 분석 PDF 다운로드 및 한글 표시
5. 파일럿 종료 후 백업 생성과 복구 확인
